# @primer/cli

Scaffold an AI-ready project in one command.

## Usage

```bash
npx @primer/cli init
```

## What it generates

- `AGENTS.md` — universal agent entry point
- `.cursor/` — Cursor rules and slash commands (if selected)
- `.claude/` — Claude Code conventions and commands (if selected)
- `docs/` — canonical architecture and pattern docs
- Full TypeScript + ESM project setup

## Requirements

- Node.js 20+
- Git
